# Note Share Web Application

This application is built as a path of my MERN stack learning journey. It is along with the authentication and authorization processes.

## Usages

Users can view the notes created by others along with the creator name. Yet, it is complusory to login to create your own note. Authorized users can edit their profile, change credentials, edit and delete the notes they created.

## Live Demo

You can try my application : [here](https://app.noteshare.page.gd/)

## Tech Stack

### FrontEnd

- React (with Vite)
- React Router
- React Redux
- React Hook Form
- Zod Validation
- Tailwind CSS
- Axios
- Toastify (for alerts)

### Backend

- Node js
- Express
- MongoDB (mongoose)
- cors
- jwt
- bcrypt
- dotenv
